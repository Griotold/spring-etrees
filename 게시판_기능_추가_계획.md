## 🎯 현재 진행: 첨부파일 기능 (S3 연동)

### 구현 단계
1. **AWS S3 설정**
   - S3 버킷 생성
   - IAM 사용자 생성 및 권한 설정
   - application.yml에 S3 설정 추가

2. **파일 엔티티 설계**
   - File 엔티티 생성 (파일명, 원본파일명, 크기, 확장자, S3 URL 등)
   - Board와 File 연관관계 설정 (1:N)

3. **파일 업로드 서비스 구현**
   - FileUploadService 생성
   - S3에 파일 업로드/삭제 로직
   - 파일 검증 (크기, 확장자)

4. **화면 연동**
   - 게시글 작성/수정 폼에 파일 업로드 UI
   - 파일 목록 표시
   - 이미지 미리보기

5. **테스트**
   - 파일 업로드/다운로드 테스트
   - 게시글 삭제 시 파일 연동 삭제 테스트
